How to Add a New Moment

Since this is a static website hosted on GitHub Pages, directly "uploading" from the browser isn't possible. Here's the simple process you'll follow to add new photos and their details:

  1. 1. Upload Image to Your GitHub Repository

    First, manually upload your photo file (e.g., `my-new-moment.jpg`) to your GitHub repository. It's highly recommended to create an `images/` folder in your repository's root and place all your photos there for organization.

  2. 2. Edit Your `index.html` File

    Open your `index.html` file in a text editor (or directly on GitHub's website). Scroll down to the `

    ` section, which contains all the "Moment Cards."

  3. 3. Add a New Moment Card

    Copy one of the existing `

    ...
    ` card structures and paste it to create a new one. Then, carefully update the following:

    • `src="images/your-new-photo.jpg"`: **Change this to the exact path of the image you just uploaded.**
    • `alt="A brief description of your photo"`: Add descriptive text for accessibility.
    • `
      Your Moment Title Here
      `: Give your moment a catchy title.
    • `

      Write a detailed description of this special memory.

      `: Elaborate on the moment.
    • `Date of the Moment`: Specify when this moment happened.
  4. 4. Commit Changes and Publish

    Save your `index.html` file. Commit these changes to your GitHub repository (e.g., with a commit message like "Add new farewell party photos"). GitHub Pages will automatically detect the changes and rebuild your website, making your new moment visible online!

This manual process is excellent for understanding how static websites work and for honing your HTML/CSS skills.
